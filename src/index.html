Home What is this?

DeckTape.io converts HTML presentations into PDFs.

You give it the address of an HTML presentation on the web, and it will generate a PDF version of that presentation. It's that simple.

DeckTape.io is powered by DeckTape, a flexible, high-quality PDF export tool. This means that DeckTape.io can handle presentations in many different formats.

You can find example presentations on the DeckTape project page.

To use DeckTape.io, just put a URL of a presentation into the box below and click "Convert!". DeckTape.io will work on it for a bit and give you a download link. Enjoy!
